# This folder contains files related to TS-DAR tutorials for Alanine Dipeptide
1. Python notebook for alanine dipeptide: (./Alanine_dipeptide.ipynb)
2. Featurized MD trajectories are available from Zenodo: [https://zenodo.org/records/16922765/files/Alanine_dipeptide.zip?download=1]

Note: the base_dir is set to './data/' in the notebook. This means that the data should be in the same directory as your notebook. If you decide to store the data elsewhere, make sure you change the path before running the code.
